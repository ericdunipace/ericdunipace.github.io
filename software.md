## R packages
I currently do most of my coding in R with some C++ sprinkled in thanks to [Rcpp.](https://dirk.eddelbuettel.com/code/rcpp.html)

* [**approxOT:**](https://github.com/ericdunipace/approxOT) approximate and exact optimal transport algorithms with linkable C++ header files <!-- [[CRAN](https://CRAN.R-project.org/package=approxOT ), [GitHub](https://github.com/ericdunipace/approxOT)] -->
* [**causalOT:**](https://github.com/ericdunipace/causalOT) using Optimal Transport methods to perform weighting for causal inference <!--[[CRAN](https://CRAN.R-project.org/package=causalOT ), [GitHub](https://github.com/ericdunipace/causalOT)] -->
* [**RcppCGAL:**](https://github.com/ericdunipace/RcppCGAL) header files from the [CGAL](https://www.cgal.org) library for use in R <!--[[CRAN](https://CRAN.R-project.org/package=RcppCGAL ), [GitHub](https://github.com/ericdunipace/RcppCGAL)] -->
* [**WpProj:**](https://github.com/ericdunipace/WpProj) Wasserstein projections for linear models 


And packages under development include

* **graphTest:** Using nearest neighbor graphs and Hamiltonian paths to do multisample testing with multivariate data
* **pycot:** Using optimal transport methods for causal inference in Python

## Other Code
* Code to reproduce the results in the paper [*Interpretable Model Summaries Using the Wasserstein Distance*](https://arxiv.org/abs/2012.09999) can be found [here](https://github.com/ericdunipace/SLIMpaper).
* Code to reproduce the results for [*Association Between Periodontal Disease and Cardiovascular Disease (from the NHANES)*](https://pubmed.ncbi.nlm.nih.gov/35773046/) can be found [here](https://github.com/ericdunipace/periodontal).
